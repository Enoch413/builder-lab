PREP MASTER BUILDER

WORKSPACE
PREP MASTER
여러 PREP 세션을 운영용 마스터로 묶는 허브
왼쪽에서 기본 설정과 반 구성을 정리하고, 오른쪽에서 학습 세트 배정과 최종 배포 파일을 마무리합니다.
Step 01
JSON 불러오기
master session.json은 복원용이고, ROTATION 세션 JSON은 추가용입니다. ROTATION 세션을 넣기 전에 아래 반 배정을 먼저 선택하세요.
다음 ROTATION 세션 기본 배정 반
아직 불러온 파일이 없습니다.
master는 현재 설정 복원용, ROTATION 세션은 학습 세트 추가용으로 사용합니다.
Step 02
PREP 기본 설정
페이지 제목을 설정합니다.
페이지 제목
Step 03
반 설정
반은 최대 8개까지 만들 수 있고, 모든 학습 세트에서 공통으로 사용합니다.
반 이름을 먼저 정하고, 필요하면 반 비밀번호를 넣은 뒤 각 학습 세트마다 어떤 지문을 보여줄지 선택합니다.
READY TO BUILD
학습 세트 구성과 최종 다운로드는 이 영역에서 계속 정리합니다. 왼쪽 반 설정이 바뀌면 세트 배정도 같이 반영됩니다.
Step 04
학습 세트 설정
학습 세트마다 제목과 기간을 정하고, 각 반에 보여줄 지문을 따로 체크합니다. 기간이 지나면 학생 화면에서는 종료 상태로 보입니다.
Step 05
master session.json 생성
고정 PREP 페이지는 그대로 두고, 이 session.json 하나만 배포하면 여러 반과 여러 기간을 함께 운영할 수 있습니다.
