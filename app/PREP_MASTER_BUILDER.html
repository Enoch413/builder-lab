PREP MASTER BUILDER

session.json builder
01
JSON 불러오기
master session.json은 복원용이고, ROTATION 세션 JSON은 추가용입니다. ROTATION 세션을 추가할 때는 아래에서 기본 배정할 반을 먼저 체크해 주세요.
다음 ROTATION 세션 기본 배정 반
아직 불러온 파일이 없습니다.
master는 현재 설정 복원, ROTATION 세션은 새 학습 세트 추가에 사용됩니다.
02
PREP 기본 설정
페이지 제목과 공통 비밀번호를 설정합니다. 공통 비밀번호를 비워 두면 반 비밀번호만 사용합니다.
페이지 제목
공통 비밀번호(선택)
03
반 설정
반은 최대 8개까지 만들 수 있고, 모든 학습 세트에서 공통으로 사용합니다.
반 이름과 반 비밀번호를 먼저 정한 뒤, 아래 학습 세트마다 어떤 지문을 보여줄지 체크해 주세요.
04
학습 세트 설정
학습 세트마다 제목과 기간을 정하고, 각 반에 보여줄 지문을 따로 체크합니다. 기간이 지나면 학생 화면에서는 종료 상태로 보입니다.
05
master session.json 생성
고정 PREP 웹페이지는 그대로 두고, 이 session.json 하나만 웹에 올리면 여러 반과 여러 기간을 함께 운영할 수 있습니다.
